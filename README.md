# Audio Remux

動画ファイルの音声トラックを差し替えるMacアプリケーション。波形表示による同期調整、複数の出力形式・コーデックに対応。

## 主な機能

- 動画の音声トラックを差し替え（映像は再エンコードなし）
- 波形表示による視覚的な同期調整（ズーム・スクロール・ドラッグ操作）
- 自動同期解析（クロス相関分析、信頼度表示付き）
- 音声再生プレビュー（元音声 / 置換音声 / 両方の3モード）
- 手動オフセット調整（±0.001秒精度）
- 自動フェード（ぶつ切りノイズ対策）
- ドラッグ&ドロップによるファイル読み込み

## 対応形式

| | 形式 |
|---|---|
| 入力（動画） | MP4, M4V, MOV |
| 入力（音声） | WAV, AIFF, FLAC |
| 出力コンテナ | MP4, MKV, MOV |
| 音声コーデック | FLAC, ALAC, AAC, PCM (16/24-bit) |

## キーボードショートカット

| キー | 操作 |
|---|---|
| Space | 再生 / 一時停止 |
| Home | 波形の先頭へ |
| End | 波形の末尾へ |
| スクロールホイール | ズーム |
| Shift + スクロール | 横スクロール |

## 動作環境

- macOS 13.0以降
- FFmpeg（初回起動時に自動ダウンロード）

## ビルド

```bash
./scripts/build-app.sh
```
